# 🎯 最简单的使用方法

## ⚡ 一分钟上手

### 方法1: 双击运行（推荐）⭐⭐⭐⭐⭐

1. **双击** `每日分析.ps1`
2. **复制** 微信聊天记录（Shift+选择，Ctrl+C）
3. **粘贴** 到打开的记事本（Ctrl+V）
4. **保存** 并关闭记事本（Ctrl+S）
5. **完成**！自动生成报告

就这么简单！🎉

---

### 方法2: 命令行（适合高级用户）

```powershell
# 1. 复制微信聊天到 my_chat.txt
# 2. 运行
python main.py --file my_chat.txt
# 3. 查看 reports 目录下的报告
```

---

## 📝 复制微信聊天的技巧

### Windows 微信

```
1. 打开聊天窗口
2. 点击第一条要复制的消息
3. 按住 Shift，滚动到最后
4. Shift + 点击最后一条
5. Ctrl + C 复制
6. Ctrl + V 粘贴到记事本
```

### 批量选择技巧

- **连续选择**: Shift + 点击
- **单个选择**: Ctrl + 点击
- **全选当天**: Ctrl + A (在聊天窗口)

## 📊 查看报告

报告保存在 `reports` 目录：

```
reports/
  ├─ report_20251105_235611.md  ← 最新报告
  ├─ messages_20251105_235611.txt  ← 原始消息
  └─ ...
```

用任何文本编辑器或 Markdown 查看器打开即可。

## ❓ 遇到问题？

### 问题：找不到 Python
**解决**：确保已安装 Python（运行 `python --version`）

### 问题：API 连接失败
**解决**：检查网络，或更换 API Key

### 问题：中文乱码
**解决**：保存文件时选择 UTF-8 编码
