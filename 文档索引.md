## 🛠️ Docker 工具脚本

所有脚本位于 `docker/` 目录

### 常用脚本

| 脚本 | 功能 | 何时使用 |
|------|------|----------|
| **一键启动.ps1** | ⭐ 自动完成所有步骤 | 首次使用或想省事 |
| start.ps1 | 启动服务 | 每天开始工作 |
| stop.ps1 | 停止服务 | 每天结束工作 |
| status.ps1 | 查看状态 | 想知道服务状态 |

### 管理脚本

| 脚本 | 功能 | 何时使用 |
|------|------|----------|
| install.ps1 | 安装镜像 | 首次使用 |
| init-db.ps1 | 初始化数据库 | 首次使用或重置 |
| logs.ps1 | 查看日志 | 调试问题 |
| query.ps1 | 查询数据 | 查看数据库 |
| backup.ps1 | 备份数据 | 定期备份 |
| restore.ps1 | 恢复数据 | 从备份恢复 |

## ⚡ 常用命令速查

```powershell
# 一键启动（推荐）
cd docker
.\一键启动.ps1

# 日常使用
.\start.ps1          # 启动
.\stop.ps1           # 停止
.\status.ps1         # 状态

# 数据管理
.\init-db.ps1        # 初始化
.\backup.ps1         # 备份
.\restore.ps1        # 恢复
.\query.ps1          # 查询

# 调试
.\logs.ps1           # 日志
```
